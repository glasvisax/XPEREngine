## Features
- OpenGL-based graphics rendering
- SSAO
- Blinn-phong lighting model
- Deferred shading 
- Normal mapping
- Shadow mapping
- Bloom and HDR
- Scene Graph
- Model loading

## Prerequisites
- CMake 
- C++17 compatible compiler
- Dependencies:
  - GLFW
  - GLAD
  - GLM
  - Assimp
  - OpenGL 4.6
  - Gtest
